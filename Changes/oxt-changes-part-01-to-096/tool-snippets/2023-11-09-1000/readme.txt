-- Please do not forget to uncompress the snippets folder to the location mentioned below


folder: scriptsnippets
[Linux:] /opt/path-to-your-build/Toolset/palettes/tools/scriptsnippets/
[win:] C:/path-to-your-build/Toolset/palettes/tools/scriptsnippets/

[Linux:] /opt/path-to-your-build/Toolset/palettes/tools/
